+  **微生物的特点**

+  **微生物学的建立**
 <details>
  <summary>2点</summary>
    <ul>
    <li>巴斯德的曲径瓶实验彻底否定了自然发生学说。</li>
    <li>科赫创立病原体的基本原则：1.在同一疾病中都出现这一病原体。2.从宿主分离出这种微生物在实验条件下能培养出这种微生物。3.再用培养出的微生物感染正常健康生命体，生命体会的相同的疾病。4.再从生病的宿主身上分离出相同的微生物。</li>
    </ul> 
</details>

+  **选择培养与富集培养的定义**
<p>非数量优势微生物类群的分离纯化则可以首先通过选择培养或富集培养使其数量增加。</p>
 <details>
  <summary>2点</summary>
    <ul>
    <li>选择培养：就是通过添加抑制剂，抑制大多数微生物的生长；或选择特定的营养物，使得需要的微生物易于快速生长。</li>
    <li>富集培养：利用不同的微生物之间的生命活动特点不同，制定特定的环境条件，使仅适应于该条件的微生物旺盛生长，从而使其在群落中的数量大大增加，人们更容易从自然界中分离所需的特定微生物。</li>
    </ul> 
</details>

 <details>
  <summary>2点</summary>

    </ul> 
</details>

+ **酶作为生物催化剂有哪些特点**
 <details>
  <summary>4点</summary>
    <ul>
    <li>高效性，比普通化学催化剂效率高至少几倍。</li>
    <li>选择性，酶是具有高度选择性的催化剂，酶往往只能催化一种或一类反应，底物往往是一种或一类极为相似的物质。</li>

    <li>酶活力可调节控制。如抑制剂调节、共价修饰调节、反馈调节、酶原激活及激素控制等。</li>
    </ul>

+ **真核生物和原核生物的细胞结构的区别，至少三个区别**
 <details>
  <summary>5点</summary>
    <ul>
    <li>真核生物有由染色体、核仁、核液、双层核膜等构成的细胞核；原核细胞无核膜、核仁、故无真正的细胞核，今有由核酸集中组成的拟核。</li>
    <li>真核细胞有内质网、高尔基体、溶酶体、液泡等细胞器，原核生物没有。</li>
    <li>真核细胞有肌动蛋白，肌球蛋白等组成的微纤维系统，后者与胞质环流、吞噬作用等密切相关；而原核生物却没有这种系统，因而也没有胞质环流核吞噬作用。</li>
    <li>真核细胞的核糖体为80s，原核生物的核糖体为70s，两者在化学结构组成和形态结构上都有明显的区别。</li>
    <li>原核生物功能上与线粒体相当的结构是质膜内褶形成的结构，但后者没有自己的基因组，也没有自己的特定合成系统。真核生物的植物含叶绿体，它们都是双层膜包裹，也有自己特有的基因组和合成系统，与光合磷酸化相关的电子传递系统位于由叶绿体的内膜内褶形成的片层上。原核生物中的蓝细菌的光合细菌，虽然也具有光合作用的膜结构，称之为类囊体，散布于细胞质中，未被双层膜包裹，不形成叶绿体。</li>
    </ul> 
    </details>

+ **什么是选择压力，如何理解进化压力？**
 <details>
  <summary>1点</summary>
    <ul>
    <li>选择压力又称为进化压力，指外界施加给一个生物进化过程的压力，从而改变该过程的前进方向，所谓达尔文的自然选择，或者物竞天择、适者生存、即自然界施与生物体选择压力从而使得适应自然环境者得以存活和繁衍。</li>
    </ul>

+ **大肠杆菌和枯草芽孢杆菌如何区分，至少说三种方法** 
    <details>
  <summary>3点</summary>
    <ul>
    <li>枯草是革兰氏阳性菌，染色后为紫色，大肠杆菌是革兰氏阴性菌，染色后为红色。</li>
    <li>枯草芽孢杆菌能产生芽孢，大肠杆菌不能产生芽孢，镜检检查；枯草芽孢杆菌会形成芽孢，且菌体较大；而大肠杆菌不能形成芽孢，菌体很小。</li>
    <li>枯草芽孢杆菌是好氧菌，需要氧气才能生长，大肠杆菌是兼性厌氧菌，可以在无氧条件下生长。</li>
    </ul>
